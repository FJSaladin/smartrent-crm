# SmartRent CRM

Plataforma CRM para la gestión de alquileres residenciales.

## Estado del proyecto
Proyecto en fase inicial de configuración.

## Tecnologías
- Frontend: React.js
- Backend: Node.js + Express
- Base de datos: MongoDB (planificado)
- Autenticación: JWT (planificado)
- IA: OpenAI API (planificado)

## Módulos creados
- Login (frontend)
- Autenticación (estructura backend)
